# \## 🔐 Advanced Security Feature (New Upgrades)

# To align with high-standard cybersecurity frameworks, we implemented an in-memory \*\*Rate Limiting \& Brute-Force Protection\*\* mechanism:

# \* \*\*Account Lockout:\*\* If a user (or bot) inputs incorrect credentials \*\*5 consecutive times\*\*, the session temporarily locks out the sign-in action.

# \* \*\*Dynamic Countdown Timer:\*\* Real-time JavaScript logic is integrated seamlessly into the existing glassmorphic login interface to display a \*\*60-second countdown\*\*, disabling the login submission until the cooldown safely expires without altering DB resources.

# \## ⚙️ Core System Architecture \& Modules

# 

# \### 1. User Authentication \& Access Control

# \* \*\*Guest Landing View:\*\* Both regular customers and admins start from a unified public guest view.

# \* \*\*Role-Based Redirection:\*\* Upon successful login, the system automatically checks roles:

# &#x20; \* Redirects `ADMIN` credentials safely to the dedicated Premium Workspace management panel.

# &#x20; \* Redirects `CUSTOMER` accounts to the client-side interaction portal.

# \* \*\*Enforced Session Validation:\*\* Major security tweaks like changing an account's Email or Password will invalidate current sessions and securely force a manual re-login.

# 

# \### 2. Client Portal (User Features)

# \* \*\*Interactive Navigation:\*\* Access to public tabs: `Home`, `Menu`, `Event`, and `Our Journey` stories.

# \* \*\*Dynamic Event Registration:\*\* Guests can view standard menus and current events, but are structurally prompted to instantly Login/Sign up before registering or joining active store events.

# \* \*\*Account Settings Management:\*\* Self-service tab allowing users to modify secure credentials on the fly.

# 

# \### 3. Premium Admin Workspace (Control Features)

# \* \*\*Structured Menu Management (CRUD):\*\* Admin can smoothly insert, preview, edit, or delete items categorized flawlessly across 5 distinct beverage/bakery classifications.

# \* \*\*Full-Cycle Event Administration:\*\*

# &#x20; \* Complete CRUD capabilities on all promotional store events.

# &#x20; \* Update live states to mark old events as \*\*"Complete Event"\*\*.

# \* \*\*Smart Registration Monitoring:\*\*

# &#x20; \* Access to high-level participant directories with custom search capabilities (Search by Name, Email, or Phone).

# &#x20; \* Advanced filtering tools to audit who registered for specific slots.

# \* \*\*Data Export \& Reporting:\*\* Direct capability to download and export live registration databases cleanly into standardized \*\*Excel (.xlsx)\*\* sheets.
